# ネットワーク構成プロジェクト - 詳細タスクリスト

**プロジェクト目標**: Kateクラウド + Variosecure共存構成の詳細要件確認
**最終成果物**: SCSK との打ち合わせ資料 + 質問リスト
**主導者**: 塩田知博（決裁）
**実行者**: 丹治統

## **フェーズ1: 情報収集・外部確認（本日〜明日）**

### **Task 1: Kateクラウド仕様確認**
**目的**: Kate側の技術要件を正確に把握
**実行者**: 丹治統
**確認先**: Kate公式ドキュメント / 既存資料 / Kate営業

- [ ] 1-1. Kate ソケット（アクセスポイント）の仕様書を確認
  - モデル、対応OS、接続方法
  - Wi-Fi規格、同時接続数など

- [ ] 1-2. ファイルサーバーとの接続要件を確認
  - NAS との接続可能性
  - Windows ファイルサーバーとの接続可能性
  - ドメイン認証（AD）への対応

- [ ] 1-3. モバイル端末からの接続方法を確認
  - VPN 不要で直接接続可能か
  - 回線要件（Wi-Fi のみか、別途LTE回線が必要か）

- [ ] 1-4. ゲスト回線対応可否を確認
  - ゲスト用アカウント・SSID への対応
  - セキュリティ分離

- [ ] 1-5. SECOM情報統合の可能性を確認
  - Variosecure からの SECOM 情報と Kate からの情報を統合できるか
  - データフォーマット、API 対応

**納期**: 本日中（Kate に問い合わせ済みなら回答待ち）

- [ ] 2-4. 導入スケジュール
  - 回線手配までの期間
