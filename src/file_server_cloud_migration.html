Googleドライブは代替になるか

直面する3つの壁

  • 排他制御の欠如: Excel等を複数の人が開くと競合コピーが発生する。
  • パス参照切れ: リンクされているファイル（ブック間参照等）が壊れる。
  • 権限の乖離: 既存ADの「部署/課」単位のきめ細かなACL構造と整合しない。

結論：一般的な事務文書や共有資料には適しているが、基幹業務で使用する「複雑なExcel」の共有場所としては不十分。

現実的な解：Azure Files

ADクラウド化とセットで導入することで、「ユーザー体験を変えずに物理サーバーを消す」ことが可能です。
