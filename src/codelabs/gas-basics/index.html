このコースで学ぶこと

このハンズオンでは、Google Apps Script（GAS）の基礎スキルを体系的に習得します。

「GAS 4行入門」でプログラムを動かす体験ができた方が次のステップとして取り組む内容です。

学習内容

  • マクロの記録と編集
  • カスタム関数の作成
  • スプレッドシート・シート・セルの操作
  • データ操作とカスタムメニュー
  • セルの書式設定（色・太字・罫線）

前提条件

  • Googleアカウントを持っていること
  • Google スプレッドシートの基本操作ができること
  • 「GAS 4行入門」を完了していること（推奨）

所要時間

約 60 分

無料で学べます
GASはGoogleアカウントのみで使えます。クレジットカード登録は不要です。

マクロとは

マクロは「一連の操作を記録したもの」です。毎回同じ手順でやっている作業を自動化できます。

例えば「毎週月曜日にヘッダー行を太字にして背景色を付ける」作業をマクロにすると、ボタン1つで完了します。

実習：ヘッダー行を整えるマクロを作る

ステップ 1：スプレッドシートを準備する

  1. sheets.new にアクセスして新しいスプレッドシートを作成
  2. A1セルに「名前」、B1セルに「部署」、C1セルに「入社年」と入力
  3. A2〜C4に適当なデータを入力しておく

ステップ 2：マクロを記録する

  1. 1行目全体（行番号「1」）をクリックして選択
  2. メニューから「拡張機能」→「マクロ」→「マクロを記録」をクリック
  3. 画面下部に「マクロを記録中...」と表示されることを確認
  4. ツールバーの「塗りつぶしの色」で背景色を濃い青にする
  5. 「フォントの色」で文字色を白にする
  6. 「太字」ボタンで太字にする
  7. 「表示」→「固定」→「1行」をクリックして1行目を固定
  8. 画面下部の「保存」をクリック
  9. マクロ名に「Header」と入力して「保存」

相対参照 vs 絶対参照
マクロ記録時に「相対参照を使用」をONにすると、現在選択しているセルからの相対位置に対して操作します。OFFの場合は常に同じセルに対して操作します。ヘッダー整形のような固定位置の操作はOFF（絶対参照）が適しています。

ステップ 3：マクロを実行してみる

  1. ヘッダーの書式を一度手動で元に戻す（色を削除など）
  2. 「拡張機能」→「マクロ」→「Header」をクリック
  3. 「承認」を求められたら「続行」→自分のアカウントを選択→「許可」
  4. 書式が自動で適用されることを確認

記録したマクロを見てみる

マクロを記録すると、裏側でJavaScriptコードが自動生成されています。これを編集することで、より細かい制御ができます。

ステップ 1：スクリプトエディタを開く

  1. 「拡張機能」→「Apps Script」をクリック
  2. スクリプトエディタが新しいタブで開く
  3. 左側の「macros.gs」または「コード.gs」をクリック

先ほど記録したマクロのコードが表示されます。おおむね以下のような内容になっているはずです：

function Header() {
  var spreadsheet = SpreadsheetApp.getActive();
  spreadsheet.getActiveRangeList()
    .setBackground('#1a237e')
    .setFontColor('#ffffff')
    .setFontWeight('bold');
  spreadsheet.getActiveSheet().setFrozenRows(1);
}

コードを手で書き直す

自動生成されたコードは冗長なことがあります。整理して書き直してみましょう。

ステップ 2：コードを書き直す

既存のコードを以下に書き換えてください：

function Header() {
  var sheet = SpreadsheetApp.getActiveSheet();
  var headerRange = sheet.getRange("1:1");

  headerRange.setBackground("#4c1130");
  headerRange.setFontColor("#ffffff");
  headerRange.setFontWeight("bold");

  sheet.setFrozenRows(1);
}

ステップ 3：保存して実行

  1. Ctrl+S（またはディスクアイコン）で保存
  2. 実行ボタン（▶）をクリック
  3. スプレッドシートに戻って結果を確認

ポイント
sheet.getRange("1:1") は「1行目全体」を意味します。getRange("A1:C1") のようにA1形式でも指定できます。

カスタム関数とは

GASで定義した関数を、Excelの =SUM() のようにセルの中で使えます。これを「カスタム関数」と呼びます。

通常のApps Scriptの関数と違い、カスタム関数はスプレッドシートのセルに直接入力して使います。

実習：単位変換関数を作る

ステップ 1：関数を書く

スクリプトエディタに以下のコードを追加してください（Header() 関数の下に追記）：

/**
 * 円をドルに変換します。
 * @param {number} yen 変換する円の金額
 * @return {string} ドルに変換した金額
 * @customfunction
 */
function YENTODOLLAR(yen) {
  var rate = 150; // 為替レート（参考値）
  var dollars = yen / rate;
  return "$" + dollars.toFixed(2);
}

@customfunction アノテーション
JSDocコメントに @customfunction を書くと、スプレッドシートでセル入力時に自動補完が表示されるようになります。

ステップ 2：スプレッドシートで使う

  1. コードを保存（Ctrl+S）
  2. スプレッドシートに戻る
  3. 空きセルに =YENTODOLLAR(10000) と入力
  4. $66.67 のような結果が表示されれば成功

ステップ 3：応用 — 税込み計算関数

同じ要領で、税込み価格を計算する関数も作ってみましょう：

/**
 * 税抜き価格から税込み価格を計算します。
 * @param {number} price 税抜き価格
 * @param {number} rate 消費税率（省略時は0.1）
 * @return {number} 税込み価格
 * @customfunction
 */
function TAXIN(price, rate) {
  if (!rate) rate = 0.1;
  return Math.round(price * (1 + rate));
}

セルに =TAXIN(1000) と入力すると 1100 が返ります。

4つの主要クラス

GASでスプレッドシートを操作するときは、4つのクラスの階層を理解することが重要です。

クラス 何を表すか 取得方法（例）
SpreadsheetApp サービス全体への入口 （常にここから始める）
Spreadsheet ファイル1つ（.xlsxに相当） SpreadsheetApp.getActiveSpreadsheet()
Sheet シート1枚（タブ1つ） spreadsheet.getActiveSheet()
Range セルのまとまり sheet.getRange("A1:C3")

実習：データを読み書きする

ステップ 1：スプレッドシートの名前を変更する

function renameSpreadsheet() {
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  ss.rename("GAS 基礎 実習ファイル");
}

ステップ 2：セルのデータを読み取る

function readData() {
  var sheet = SpreadsheetApp.getActiveSheet();

  // 1つのセルを読む
  var a1Value = sheet.getRange("A1").getValue();
  Logger.log("A1の値: " + a1Value);

  // 複数セルをまとめて読む（2次元配列で返ってくる）
  var values = sheet.getRange("A1:C3").getValues();
  Logger.log(values);
}

getValues() は2次元配列
getRange("A1:C3").getValues() は行×列の2次元配列を返します。values[0][0] がA1、values[0][1] がB1、values[1][0] がA2です。

ステップ 3：セルにデータを書き込む

function writeData() {
  var sheet = SpreadsheetApp.getActiveSheet();

  // 1つのセルに書き込む
  sheet.getRange("E1").setValue("テスト");

  // 複数セルにまとめて書き込む（個別より高速）
  var data = [
    ["山田 太郎", "営業部", 2020],
    ["佐藤 花子", "総務部", 2021],
    ["鈴木 一郎", "IT部",   2019]
  ];
  sheet.getRange(2, 1, 3, 3).setValues(data);
}

ステップ 4：最終行を取得する

function getLastRow() {
  var sheet = SpreadsheetApp.getActiveSheet();
  var lastRow = sheet.getLastRow();
  Logger.log("データが入っている最終行: " + lastRow);

  // 最終行の次の行にデータを追加する
  sheet.getRange(lastRow + 1, 1).setValue("新しいデータ");
}

実行後、「表示」→「ログ」で結果を確認してください。

カスタムメニューとは

スプレッドシートのメニューバーに独自のメニューを追加できます。チームで使う共有スプレッドシートに「一括処理」ボタンを設置するのに便利です。

実習：「データ整形」メニューを作る

ステップ 1：onOpen() 関数を書く

onOpen() はスプレッドシートを開くたびに自動実行される「シンプルトリガー」です。

function onOpen() {
  var ui = SpreadsheetApp.getUi();
  ui.createMenu("データ整形")
    .addItem("ヘッダーを整形", "Header")
    .addSeparator()
    .addItem("データを書き込む", "writeData")
    .addItem("最終行を確認", "getLastRow")
    .addToUi();
}

ステップ 2：スプレッドシートを再読み込みする

  1. コードを保存
  2. スプレッドシートのタブを再読み込み（F5）
  3. メニューバーに「データ整形」が追加されていることを確認
  4. 各メニュー項目が正しく動作することを確認

ステップ 3：セパレーターとサブメニュー

function onOpen() {
  var ui = SpreadsheetApp.getUi();
  ui.createMenu("データ整形")
    .addItem("ヘッダーを整形", "Header")
    .addSeparator()
    .addSubMenu(
      ui.createMenu("データ操作")
        .addItem("データを書き込む", "writeData")
        .addItem("最終行を確認", "getLastRow")
    )
    .addToUi();
}

注意
onOpen() は「シンプルトリガー」のため、実行できる操作に制限があります。ファイルの作成・削除や外部サービスへのアクセスなど、高い権限が必要な処理は「インストーラブルトリガー」を使う必要があります。

書式設定でスプレッドシートを見やすくする

GASでは、色・太字・罫線・数値フォーマットなどをコードで設定できます。定期的に生成するレポートの見た目を統一するのに役立ちます。

実習：レポートスタイルを適用する

ステップ 1：背景色とフォント設定

function formatReport() {
  var sheet = SpreadsheetApp.getActiveSheet();

  // ヘッダー行を整形
  var header = sheet.getRange("A1:C1");
  header.setBackground("#007272");    // 深緑の背景
  header.setFontColor("#ffffff");     // 白文字
  header.setFontWeight("bold");       // 太字

  // データ行の交互カラー（バンディング）
  var dataRange = sheet.getRange("A2:C10");
  var banding = dataRange.applyRowBanding();
  banding.setHeaderRowColor("#007272");
  banding.setFirstRowColor("#e8f5e9");
  banding.setSecondRowColor("#ffffff");
}

ステップ 2：罫線を引く

function addBorder() {
  var sheet = SpreadsheetApp.getActiveSheet();
  var range = sheet.getRange("A1:C10");

  // 外枠と内側に罫線を引く
  range.setBorder(
    true,   // 上
    true,   // 左
    true,   // 下
    true,   // 右
    true,   // 縦の内側
    true,   // 横の内側
    "#cccccc",
    SpreadsheetApp.BorderStyle.SOLID
  );
}

ステップ 3：数値フォーマットと列幅の自動調整

function formatNumbers() {
  var sheet = SpreadsheetApp.getActiveSheet();

  // C列を「年」として整数表示
  sheet.getRange("C2:C100").setNumberFormat("0");

  // 日付フォーマット（D列が日付の場合）
  // sheet.getRange("D2:D100").setNumberFormat("yyyy/MM/dd");

  // 列幅を自動調整（A〜C列 = 1〜3列）
  sheet.autoResizeColumns(1, 3);
}

ステップ 4：まとめて実行するマスター関数

function applyAllFormats() {
  formatReport();
  addBorder();
  formatNumbers();
  SpreadsheetApp.flush(); // 変更を即時反映
  SpreadsheetApp.getUi().alert("書式設定が完了しました！");
}

SpreadsheetApp.flush()
GASはパフォーマンスのため変更をまとめて送信します。flush() を呼ぶと変更が即時反映されます。ループの途中で画面を更新したいときに使います。
