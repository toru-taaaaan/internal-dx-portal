Google Apps Script は 完全無料 です。クレジットカード登録不要、Googleアカウントさえあれば今すぐ始められます。

Google Apps Script（略称: GAS）は、Google が提供する 無料のプログラミング環境 です。スプレッドシート・Gmail・カレンダー・ドライブといった Google サービスを、コードで自動的に動かすことができます。

GAS の3つの特徴

  • 完全無料：Googleアカウントのみで使える。カード登録なし
  • ブラウザで完結：インストール不要。ブラウザ上でコードを書いて実行できる
  • Googleと直接つながる：スプレッドシート・Gmail・カレンダーを公式APIで操作できる

このCodelabで作るのは「住所を入力したら、地図をメールで送ってくれるプログラム」です。スプレッドシート・Google Maps・Gmail の3つを4行でつなぎます。

まず、プログラムが読み取るデータを入力するスプレッドシートを作ります。

手順 1：新しいスプレッドシートを作る

  1. ブラウザで sheets.new を開く（または Google ドライブから「新規」→「スプレッドシート」）
  2. 新しいスプレッドシートが開いたことを確認する

手順 2：住所を入力する

  1. セル A1（左上の最初のマス）をクリックする
  2. 住所を入力する

入力例：

東京都千代田区丸の内1-1-1

住所は日本語でも英語でも構いません。実際に存在する住所を入力してください。架空の住所だと地図が正しく生成されません。

手順 3：スプレッドシートに名前をつける

  1. 左上の「無題のスプレッドシート」をクリック
  2. 「地図送信テスト」などわかりやすい名前に変更する
  3. Enter キーで確定

ここまでできたら、次のステップへ進みます。

作成したスプレッドシートから Apps Script エディタを開きます。エディタはコードを書いて実行するための画面です。

エディタの開き方

  1. スプレッドシートの上部メニューから 「拡張機能」 をクリック
  2. 表示されたメニューから 「Apps Script」 をクリック
  3. 新しいタブでエディタが開く

「拡張機能」が見当たらない場合は、画面を広げるか、横にスクロールしてみてください。

既存のコードを削除する

エディタには最初から以下のコードが書かれています：

function myFunction() {

}

これを すべて選択して削除 してください（Ctrl+A → Delete）。次のステップで新しいコードを書きます。

エディタに以下のコードを入力します。これが今回のプログラムの全体です。

コードを入力する

以下のコードをエディタに コピー&ペースト してください：

function sendMap() {
  var address = SpreadsheetApp.getActiveSheet().getRange("A1").getValue();
  var map = Maps.newStaticMap().addMarker(address);
  GmailApp.sendEmail(Session.getActiveUser().getEmail(), "地図を送ります", "住所: " + address, {attachments: [map]});
}

送信先メールアドレスは Session.getActiveUser().getEmail() で自分自身のアドレスを自動取得しています。別の宛先に送りたい場合は、ダブルクォートの中に直接メールアドレスを書いてください（例: "tanaka@example.com"）。

4行それぞれの意味

行

コード

意味

1行目

SpreadsheetApp...getValue()

スプレッドシートのA1セルの値（住所）を読み取る

2行目

Maps.newStaticMap()...

その住所にピンを立てた地図画像を生成する

3行目

コードを保存する

  1. キーボードで Ctrl + S（Mac は Cmd + S）を押す
  2. または上部の「💾 保存」アイコンをクリックする
  3. プロジェクト名を聞かれたら「地図送信」などと入力して OK

保存できたら、プログラムを実行します。

手順 1：実行する関数を選ぶ

  1. エディタ上部の関数選択ドロップダウン（「関数を選択」と表示されている部分）をクリック
  2. 「sendMap」 を選ぶ

手順 2：実行ボタンを押す

  1. 上部の 「▶ 実行」 ボタン（三角形のアイコン）をクリックする

手順 3：権限を許可する

初回実行時は、プログラムがGoogleサービスにアクセスする許可を求めてきます。

  1. 「承認が必要です」というダイアログが表示される
  2. 「権限を確認」 をクリック
  3. Googleアカウントの選択画面が出たら、使用中のアカウントを選ぶ
  4. 「このアプリはGoogleで確認されていません」と表示されたら → 「詳細」 をクリック → 「（プロジェクト名）（安全ではないページ）に移動」 をクリック
  5. アクセス許可の一覧が表示されたら 「許可」 をクリック

「このアプリはGoogleで確認されていません」という警告は、自分で作ったスクリプト なので正常です。他人が作ったスクリプトを実行するときは内容をよく確認してください。

手順 4：メールを確認する

  1. 実行が完了したら（エディタ下部に「実行完了」と表示される）
  2. Gmail を開く
  3. 「地図を送ります」という件名のメールが届いていることを確認する
  4. 添付画像を開くと、A1に入力した住所にピンが立った地図が表示される

メールが届くまで1〜2分かかる場合があります。届かない場合は「迷惑メール」フォルダも確認してください。
