このハンズオンでは、Googleの無料BIツール「Looker Studio」を使って、Googleスプレッドシートのデータをインタラクティブなダッシュボードに変換する方法を学びます。

プログラミング不要。ドラッグ＆ドロップで本格的なデータ可視化が作れます。

このコースで学ぶこと

  • Looker Studio の基本概念（データソース・グラフ・ページ）
  • Googleスプレッドシートをデータソースに接続する方法
  • 棒グラフ・円グラフ・表を追加する方法
  • フィルターと日付範囲コントロールの設定方法
  • 完成したダッシュボードを共有・公開する方法

対象者

  • Excelやスプレッドシートで集計しているが、グラフ化が面倒な方
  • 上司や関係者に見やすい報告ダッシュボードを作りたい方
  • プログラミング経験不要

所要時間

約 45分

使用するもの

  • Google アカウント
  • Google スプレッドシート（このコースで作成します）
  • Looker Studio（lookerstudio.google.com）— 無料

Looker Studio（旧Google Data Studio）は、Googleが無料で提供するデータ可視化・BIツールです。

できること

機能 説明
データ接続 Googleスプレッドシート・BigQuery・Google Analytics・CSVなど800以上のソースに接続
グラフ作成 棒グラフ・折れ線・円グラフ・表・地図・スコアカードなど多彩なビジュアル
フィルター・日付範囲 閲覧者がダッシュボード上でデータを絞り込める
自動更新 データソースが更新されると自動でグラフも更新される
共有 URLで共有・PDFダウンロード・Googleドライブ経由での権限管理

Excelグラフとの違い

比較項目 Excelグラフ Looker Studio
データ更新 手動で貼り直し 自動更新
共有 ファイル送付 URLで即共有
インタラクション 静的 クリックでフィルター
費用 Office ライセンス必要 完全無料
活用シーン：毎週上長に送るKPIレポート、月次の経費集計、プロジェクト進捗ダッシュボード。スプレッドシートをいじるだけでダッシュボードが自動更新されるので、報告資料の作り直しが不要になります。

まず、Looker Studio に接続するサンプルスプレッドシートを作成します。

Googleスプレッドシートを開く

  1. sheets.google.com を開く
  2. 「新しいスプレッドシート」をクリック
  3. シート名を「月次売上データ」に変更

サンプルデータを入力する

以下のデータをA1セルから入力してください（ヘッダー行を含む）。

A: 年月 B: 部門 C: 売上（万円） D: 費用（万円）
2026-01情報システム12085
2026-01総務9572
2026-01営業310180
2026-02情報システム13588
2026-02総務10275
2026-02営業290170
2026-03情報システム15090
2026-03総務11078
2026-03営業340195
注意：A列（年月）は YYYY-MM 形式で入力してください。日付として認識されないと、後で時系列グラフが正しく表示されません。

スプレッドシートのURLをコピー

  1. ブラウザのアドレスバーからスプレッドシートの URL をコピーしておく

次のステップで Looker Studio からこのシートに接続します。

Looker Studio を開いて、作成したスプレッドシートを接続します。

Looker Studio を開く

  1. lookerstudio.google.com を開く
  2. Googleアカウントでログイン
  3. 「空のレポート」をクリック

データソースを追加する

  1. 「データのソース」パネルが開くので「Googleスプレッドシート」を選択
  2. 先ほど作成した「月次売上データ」シートを選択
  3. 「追加」をクリック
  4. 「レポートに追加」をクリック

フィールドを確認する

Looker Studio がスプレッドシートのヘッダーを自動認識し、「フィールド」として取り込みます。

フィールド名 種別 説明
年月 ディメンション データを分類・グループ化する軸（日付・文字列など）
部門 ディメンション データを分類・グループ化する軸
売上（万円） 指標 集計・計算される数値
費用（万円） 指標 集計・計算される数値
ディメンション vs 指標：グラフのX軸や色分けに使うのがディメンション（年月・部門・カテゴリなど）、高さや大きさに使うのが指標（数値）です。

3種類のグラフを追加して、基本的な操作を体験します。

① 棒グラフ：月別売上

  1. 上部メニュー「グラフを追加」→「棒グラフ」をクリック
  2. キャンバス上でドラッグしてサイズを決める
  3. 右の「グラフ設定」パネルで：
    • ディメンション: 「年月」
    • 指標: 「売上（万円）」

月ごとの売上が棒グラフで表示されます。

② 円グラフ：部門別売上構成

  1. 「グラフを追加」→「円グラフ」をクリック
  2. キャンバスの空きスペースに配置
  3. 設定パネルで：
    • ディメンション: 「部門」
    • 指標: 「売上（万円）」

部門ごとの売上割合が円グラフで表示されます。

③ 表：詳細データ

  1. 「グラフを追加」→「表」をクリック
  2. キャンバスに配置
  3. 設定パネルで：
    • ディメンション: 「年月」「部門」
    • 指標: 「売上（万円）」「費用（万円）」

グラフの見た目を整える

右パネルの「スタイル」タブで色・フォント・境界線などを変更できます。会社のブランドカラーに合わせましょう。

便利な機能：グラフをクリックすると他のグラフが連動してフィルタリングされます（クロスフィルタリング）。「グラフを設定」→「インタラクション」→「フィルタとして適用」を有効にすることで使えます。

閲覧者がダッシュボード上でデータを絞り込めるコントロールを追加します。

日付範囲コントロールの追加

  1. 「コントロールを追加」→「日付範囲コントロール」をクリック
  2. ダッシュボードの上部に配置
  3. 右パネルの「コントロールのフィールド」で「年月」を選択

閲覧時に期間を絞り込めるようになります。

ドロップダウンフィルターの追加

  1. 「コントロールを追加」→「ドロップダウン」をクリック
  2. 日付範囲コントロールの隣に配置
  3. 設定パネルで「コントロールのフィールド」→「部門」を選択

閲覧者が部門を選んでデータを絞り込めます。

フィルターの動作確認

  1. 右上の「表示」ボタンをクリックして閲覧モードへ切り替え
  2. 日付範囲を変更してグラフが更新されることを確認
  3. 部門ドロップダウンで「営業」を選択してグラフが絞り込まれることを確認
  4. 「編集」ボタンで編集モードに戻る
複数グラフへの一括適用：コントロールはページ内の全グラフに適用されます。特定のグラフだけに適用したい場合は、グラフ設定の「フィルター」から個別に設定できます。

完成したダッシュボードを関係者と共有します。

共有方法1：URLで共有

  1. 右上の「共有」ボタンをクリック
  2. 「他のユーザーと共有」を選択
  3. メールアドレスを入力、または「リンクを知っているユーザー全員」に変更
  4. 権限を「閲覧者」に設定して「送信」

共有方法2：PDF ダウンロード

  1. 「ファイル」→「PDFとしてダウンロード」をクリック
  2. ページサイズ・向きを選択して「ダウンロード」

定例報告メールへの添付に使えます。

共有方法3：スケジュール送信

  1. 「ファイル」→「メールのスケジュール設定」をクリック
  2. 宛先・送信頻度（毎日/毎週/毎月）を設定
  3. 「保存」

毎週月曜朝に自動でPDFが送られるように設定できます。定例報告の自動化に活用しましょう。

データの自動更新：スプレッドシートのデータを更新すると、Looker Studio のダッシュボードも自動で反映されます。データ入力担当者がシートを更新するだけで、全員のダッシュボードが最新状態になります。
