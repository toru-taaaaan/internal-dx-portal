このコースについて

Google Sites を使って、プログラミング不要で社内 Wiki・プロジェクトポータル・チームサイトを構築するハンズオンです。Docs・Sheets・カレンダーなど Workspace ツールを埋め込んで、情報の集約場所を30分で作れます。

学習目標

  • Google Sites でサイトを新規作成する
  • ページを追加してコンテンツを編集する
  • Docs・Sheets・スライド・カレンダーを埋め込む
  • ナビゲーションを設定して使いやすくする
  • サイトを社内公開・社外公開する

進捗・資料・議事録を1か所に集約

サイトの作成と基本設定

サイトを作成する

  1. sites.google.com を開く
  2. ＋ 新しい Google サイト をクリック
  3. テンプレートが表示される。今回は「空白」を選択

テーマとレイアウトを選択する

  1. 右側のパネルで テーマ をクリック
  2. 「Aristotle」「Luxe」「Diplomat」などから選ぶ（後から変更可）
  3. カラーや書体も選択できる

サイト名（タイトル）を設定する

  1. 左上の「無題のサイト」をクリックして変更
    例: AI Champion プロジェクトポータル
  2. ページ上部のバナー部分のタイトルも変更する
    例: AI Champion プロジェクトポータル
  3. サブタイトル欄に説明を追加
    例: AI活用推進プロジェクトの情報・資料・進捗を集約しています

バナー画像を変更する

  1. バナー部分にカーソルを合わせると 「画像を変更」 ボタンが表示される
  2. Google 提供の画像を選ぶか、自分でアップロードする

コンテンツの追加方法

テキストブロックを追加する

  1. 右側パネルの 挿入 タブをクリック
  2. テキストボックス をドラッグしてページ上に配置
  3. 見出し・本文・箇条書きなどを入力

画像を追加する

  1. 挿入 → 画像 をクリック
  2. 「アップロード」「Google ドライブから選択」「URL で指定」から選ぶ

ページを追加する

  1. 右側パネルの ページ タブをクリック
  2. 下部の ＋ 新しいページ をクリック
  3. ページ名を入力（例: 資料一覧 / 議事録 / メンバー紹介）

おすすめのページ構成例

ページ名コンテンツ
ホームプロジェクト概要・最新情報・リンク集
資料提案書・報告書の Docs/スライド埋め込み
議事録会議ごとの議事録 Docs 一覧
スケジュールGoogle カレンダー埋め込み
メンバー担当者・役割・連絡先一覧
ページ数が多くなると迷子になりやすくなります。最初は3〜5ページに絞って、必要に応じて追加するのがおすすめです。

Workspace ツールの埋め込み

Google Sites の最大の強みは、Workspace ドキュメントをそのまま表示できることです。

Google Docs を埋め込む

  1. 挿入 → ドライブ をクリック
  2. 埋め込みたい Docs ファイルを選択して「挿入」
  3. Docs が更新されると Sites 上の表示も自動で更新される
ポイント
Docs の内容が変わると Sites の表示も自動で反映されます。「最新資料はここを見て」と案内できるので、バラバラなファイル共有が不要になります。

Google スライドを埋め込む

  1. 挿入 → ドライブ → スライドファイルを選択
  2. サイズを調整して配置

Google カレンダーを埋め込む

  1. 挿入 → カレンダー をクリック
  2. 表示したいカレンダーを選択して挿入
  3. サイト上でカレンダーの予定が確認できるようになる

Google Forms を埋め込む

  1. 挿入 → ドライブ → Forms ファイルを選択
  2. サイト内でそのままフォームに回答できる

YouTube 動画を埋め込む

  1. 挿入 → YouTube
  2. 動画の URL または検索キーワードで動画を選択
埋め込みコンテンツ用途
Docsマニュアル・議事録・報告書の表示
スライド研修資料・プレゼンの閲覧
Sheets進捗表・データの埋め込み表示
カレンダープロジェクトスケジュール・締め切り管理
Formsアンケート・申込フォームの直接回答
YouTube研修動画・操作説明動画

見やすいナビゲーション構造

ナビゲーションの種類

Google Sites のナビゲーションは上部（ヘッダー）に表示されます。

ページの並び順を変更する

  1. 右側パネルの ページ タブを開く
  2. ページをドラッグ&ドロップで並び替え

サブページを作成する

  1. ページの右にある 縦3点メニュー → サブページを追加
  2. 親ページの下にネスト（階層化）される

例:

  • 資料
    • 2025年度
    • 2026年度
  • 議事録
    • 月次定例
    • 臨時会議

外部リンクをナビゲーションに追加する

  1. ページパネル下部の リンクを追加
  2. タイトルと URL を入力
  3. 例: タスク管理表（Google スプレッドシートへのリンク）
ナビゲーションは5〜7項目以内に収めると見やすくなります。階層が深すぎると迷子になるので、2階層までを目安にしましょう。

公開設定と権限管理

サイトを公開する

  1. 右上の 公開 ボタンをクリック
  2. ウェブアドレス（URL）を設定する
    例: sites.google.com/view/akiba-ai-champion
  3. 公開範囲を設定する

公開範囲の設定

設定アクセスできるユーザー用途
組織内のみ@会社ドメインのアカウント社内専用サイト（推奨）
リンクを知っている全員URL を知っている人全員取引先との共有など
全員（公開）検索エンジンにも表示される外部向けサイト
社内情報・プロジェクト資料は必ず「組織内のみ」で公開してください。誤って全体公開すると社外に情報が漏れる可能性があります。

編集権限を共有する

  1. 右上の 共有アイコン をクリック
  2. 共同編集者のメールアドレスを追加
  3. 「編集者」または「閲覧者」を選択
役割できること
オーナー全操作・公開設定・削除
編集者コンテンツ追加・編集（公開設定は変更不可）
閲覧者閲覧のみ（公開前の確認に使用）
