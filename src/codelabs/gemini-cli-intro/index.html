Gemini CLI とは

Gemini CLI は、ターミナル（コマンドライン）から直接 Gemini AI と会話できるツールです。ブラウザを開かなくても、作業中のファイルを読ませながら質問したり、コードを生成したりできます。

「AIチャットツールをターミナルに持ち込んだもの」とイメージしてください。自分のパソコン上のファイルを直接参照できるのが最大の特徴です。

このコースで学ぶこと

  • Gemini CLI のインストールと初期設定
  • 基本的なコマンドと使い方
  • ファイルを読ませて質問する
  • コードの生成と修正
  • 日常業務（要約・下書き・分析）への応用

前提条件

  • Google アカウントを持っていること
  • Node.js 18 以上がインストールされていること（次のステップで確認します）
  • ターミナル（コマンドプロンプト）を起動できること

所要時間

約 50 分

個人アカウントなら無料で使えます
Google の個人アカウントでログインすると、Gemini 2.5 Pro を無料で利用できます（60 リクエスト/分、1,000 リクエスト/日）。

Node.js のバージョンを確認する

ターミナルを開いて以下を実行してください。

node --version

v18.x.x 以上と表示されれば OK です。表示されない場合や v18 未満の場合は、nodejs.org から最新版（LTS）をインストールしてください。

Gemini CLI をインストールする

npm install -g @google/gemini-cli

インストールが完了したら、バージョンを確認します。

gemini --version

Windows の場合
コマンドプロンプトまたは PowerShell を「管理者として実行」して入力してください。

Gemini CLI を起動して認証する

  1. ターミナルで gemini と入力して起動
  2. テーマの選択画面が表示されたら好みのものを選ぶ
  3. 認証方法の選択画面が表示される
  4. 「Login with Google」 を選ぶ
  5. ブラウザが開くので、Google アカウントでログイン
  6. ターミナルに入力欄が表示されたらセットアップ完了

確認：最初の質問をしてみる

入力欄に以下を入力して Enter を押してください。

こんにちは。今日の日付と曜日を教えてください。

Gemini が返答すれば、セットアップは成功です。

終了するには
/quit と入力するか、Ctrl-C を 2 回押します。

ヘルプを表示する

Gemini CLI を起動した状態で /help と入力すると、使えるコマンドとキーボードショートカットの一覧が表示されます。

/help

主なコマンド一覧

コマンド 説明
/help コマンド一覧を表示
/clear 会話履歴をリセット
/tools 使えるツール一覧を表示
/model 使用するモデルを切り替え
/quit 終了

非インタラクティブモード（1 回きりの質問）

毎回起動しなくても、コマンドラインで直接質問できます。

gemini "Googleスプレッドシートのセル範囲を取得するGASのコードを書いてください"

ただし、このモードでは会話の続きを行うことはできません。続けて深掘りしたい場合は通常モード（gemini）を使いましょう。

モデルを指定して起動する

gemini -m "gemini-2.5-flash"

Flash モデルは応答が速く、シンプルな質問に向いています。複雑な質問には Pro（デフォルト）が適しています。

組み込みツール一覧を確認
/tools を実行すると、ファイルの読み書き・Web 検索・Shell 実行など、Gemini が自動的に使えるツールが確認できます。

@ 記法でファイルを参照する

Gemini CLI では @ファイル名 と書くことで、ローカルのファイルを読ませながら質問できます。

実習：テキストファイルを要約させる

ステップ 1：テスト用ファイルを作る

まず作業用フォルダを作って移動します。

mkdir gemini-practice
cd gemini-practice

次に、サンプルのテキストファイルを作成します。以下の内容を meeting_note.txt として保存してください。

2026年3月の全体会議 議事録

参加者：田中、鈴木、山田、佐藤

議題1：Q4売上報告
・売上は前年比105%。目標達成。
・課題：西日本エリアが前年割れ（▲8%）
・対策：来月中に西日本営業チームとミーティングを設定

議題2：新システム導入について
・4月から会計システムを刷新予定
・各部署への説明会を3月末までに実施する
・IT部（丹治）が窓口。質問は直接連絡のこと

議題3：社員旅行
・今年は10月を予定。行き先は投票で決定
・アンケートを今週中に配布予定

次回会議：4月5日（金）10:00〜

ステップ 2：Gemini CLI で要約させる

gemini を起動して以下を入力します。

@meeting_note.txt の内容を3行で要約してください。

ステップ 3：深掘り質問をする

このファイルの中で、アクションアイテム（誰かが対応すべきこと）を箇条書きにしてください。
西日本エリアの課題に対する追加の対策案を3つ提案してください。

複数ファイルの参照

複数のファイルを同時に読ませることもできます。

@report_jan.txt と @report_feb.txt を比較して、変化した点をまとめてください。

対応しているファイル形式
テキスト（.txt, .md, .csv, .json）、コード（.js, .py, .html）、PDF などが読めます。Excel や Word は直接は読めませんが、テキストに変換してから使うと便利です。

GAS コードを生成させる

Gemini CLI はコード生成が得意です。「こういう処理をするコードが欲しい」と日本語で伝えるだけで、動くコードを生成してくれます。

実習 1：スプレッドシートのコードを生成する

Google Apps Script で以下の処理をするコードを書いてください。

・スプレッドシートのA列に名前、B列にメールアドレスが入っている
・C列に「確認済み」か「未確認」をランダムに入れる処理（テスト用）
・D列に今日の日付を入れる

先ほど生成したコードを貼り付けて、改善を頼みます。

エラーを修正させる

コードがエラーになったときは、エラーメッセージごと貼り付けて聞きます。

以下のコードでエラーが出ます。原因と修正方法を教えてください。

コード：
function myFunction() {
  var sheet = SpreadsheetApp.getActiveSheet();
  var data = sheet.getRange("A1:B").getValues();
  Logger.log(data);
}

エラーメッセージ：
Range not found (line 3)

既存コードを説明させる

他の人が書いたコードや、自分で書いて忘れたコードの説明も得意です。

以下のコードが何をしているか、非エンジニアにも分かるように説明してください。

function sendWeeklyReport() {
  var ss = SpreadsheetApp.openById("xxxxx");
  var sheet = ss.getSheetByName("週次集計");
  var lastRow = sheet.getLastRow();
  var data = sheet.getRange(2, 1, lastRow - 1, 4).getValues();
  var total = data.reduce(function(sum, row) { return sum + row[3]; }, 0);
  GmailApp.sendEmail("manager@example.com", "週次レポート", "今週の売上合計: " + total + "円");
}

コードを貼り付けるコツ
コードを貼り付ける前後に説明文を入れると、Gemini が文脈を理解しやすくなります。「以下のコードを〜してください」という形式が有効です。

メール下書きを生成させる

Gemini CLI はビジネス文書の生成も得意です。

以下の状況に合わせて、ビジネスメールの下書きを作成してください。

送信者：情報システム部 丹治
宛先：全社員
件名：新しい会計システムへの移行について

内容：
・4月1日から会計システムが「〇〇クラウド会計」に変わる
・3月中に各部署で説明会を実施する（日程は後日連絡）
・移行前に現システムのデータエクスポートが必要（IT部がサポート）
・質問は内線 2345 または it-support@company.co.jp まで

丁寧なビジネス文体で、300文字程度にまとめてください。

議事録を整形させる

音声文字起こしの生テキストを整形するのも効果的です。

以下の音声文字起こしテキストを、読みやすい議事録形式に整形してください。
箇条書きを使い、アクションアイテムを最後にまとめてください。

[文字起こしテキストを貼り付け]

Excel・CSV データを分析させる

CSV ファイルを @ で読み込ませると、データ分析も可能です。

@sales_data.csv のデータを分析して、
1. 売上上位3商品
2. 最も低迷している月
3. 全体のトレンド
を教えてください。

Web ページの内容を要約させる

URL を貼り付けて要約を依頼できます（WebFetch ツールを使用）。

https://developers.google.com/apps-script/overview の内容を日本語で要約してください。

活用のコツ
「誰に向けた文章か」「何文字程度か」「どんなトーンか」を指示に入れると、より実用的な出力が得られます。

学習内容の振り返り

  • Gemini CLI をインストールして Google アカウントで認証できた
  • /help や /tools などの基本コマンドを使えた

  • GAS コードの生成・修正・説明ができた
  • メール下書き・議事録整形などの業務タスクに活用できた
