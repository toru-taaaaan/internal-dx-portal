現状と課題

ネットワーク

  • • 多摩・橋本拠点の通信速度が遅い
  • • クラウドサービス（M365等）利用時の体感速度が悪い
  • • 拠点間通信の最適化ができていない

セキュリティ

  • • UTM機能（ファイアウォール、不正侵入防御等）が不足
  • • サイバー攻撃への防御体制が脆弱
  • • 機種混在により管理画面が分断（LBO可/不可で2タイプ）

現状維持

USEN Vario Secure 継続利用

  • ⚠ ネットワーク改善なし、セキュリティ強化なし

② USEN 現状強化

高額

Vario Secure + セキュリティ強化オプション

  • • UTM機能（FW、IPS、アンチウイルス等）を追加
  • ⚠ 月額約7倍増（約15.5万円 → 約111.7万円）
  • • セキュリティは強化されるが、ネットワーク改善は限定的

③ IIJ Omnibus

有力候補

  • ✓ ネットワーク + セキュリティを統合提供
  • ✓ 8拠点一括管理、国内品質重視
  • • 月額¥722,150（ネットワーク＋FW込み）、初期費用¥2,764,850

見積待ち

Cato Networks SASE（次世代型）

  • ✓ 次世代SASE（ゼロトラスト）、グローバル対応
  • ✓ 初期費用 ¥0（Zero Touch）
  • ⏳ SCSK社から見積回答待ち
