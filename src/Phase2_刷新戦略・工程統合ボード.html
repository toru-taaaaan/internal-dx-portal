Google Sheets: GPO管理台帳 / アクセス制御管理台帳

GPO管理台帳（10シート）
読込完了 → 分析結果をsec6に反映
  • Shared Computer GPO: 14設定（WinUpdate, IE mode, RDP, Defender等）
  • Shared User GPO: スクリーンセーバー300秒、ドライブマップ S: → \\adsharesvr\public
  • IoT GPO: ドライブマップ U: → \\iotsvr\share
  • その他: FW無効化、Win11バージョン制御、TENSUITE VPN回避策
結論: ドライブマッピング（S:/U:）がEntra ID移行の最大障壁。他はIntune代替可能
アクセス制御管理台帳
読込完了 → 分析結果をsec6に反映
  • ADSHARESVR: 40+部門フォルダ（個人名ベースのフルアクセス権限）
  • HD直下: グループ監査室、コンプライアンス、経営企画室、人事部、総務部等
  • 深いネスト: 総務専用 → 取締役会 → 資料 → グループ経営会議
  • IOTSVR: 6フォルダ（sales_極秘、engineer_極秘等）IoTG+情シスのみ
結論: 個人名ベースACLのクラウド再現が大工数。Azure Files移行時の最重要検討事項
