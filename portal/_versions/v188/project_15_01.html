🛠 技術仕様・制約

IIJ (PaaS) の制約

  • OS固定: Windows Server 2022のみ。細かい設定不可。
  • 権限制限: Domain Admin権限なし (一部制限あり)。
  • RDP不可: RSAT (管理ツール) 経由のみ。
  • ベンダーロックイン: Omnibus (SD-WAN) 必須。

USEN / Link At Japan (IaaS) のメリット

  • 完全な自由度: Domain Adminフル権限付与。
  • 既存資産活用: USENは既存VSRルーター利用可 (初期コスト減)。
  • BCP対策: LAJは「東日本/西日本」のリージョン冗長化構成。
  • 拡張性: 同じ箱に「セコムサーバー」や「ファイルサーバー」を追加容易。

🏆 最終推奨: Link At Japan

理由1: 圧倒的なコストメリット

IIJと比較して5年間で約400万円の削減。Azure RI (予約インスタンス) 適用による長期的な割引効果が大きい。

理由2: BCP対策 (東西冗長化)

東日本・西日本の2拠点構成により、大規模災害時も認証基盤を維持可能。他社は東京一極集中やオプション扱い。

理由3: IaaSの柔軟性

将来的にセコムサーバーやファイルサーバーを統合する際、同じAzure VNet内にスムーズに追加・統合が可能。
